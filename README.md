## 소개

따봉도치는 Claude Code 에이전트 팀을 **픽셀아트 교실**에서 시각화하고, 토큰 사용량과 비용을 실시간으로 추적하는 macOS 앱입니다.

---

## 주요 기능

### 🏫 인터랙티브 교실

- 팀원이 픽셀 고슴도치 캐릭터로 교실에 표시
- 작업 중인 에이전트는 애니메이션으로 상태 표시
- 칠판에 팀 메시지/채팅 실시간 표시
- 게시판에 메모 노트 표시
- 낮/밤 테마 자동 전환 (오전 7시 ~ 오후 7시)
- 마우스 휠로 칠판 메시지 스크롤

### 📊 토큰 사용량 추적

- **오늘/주간** 토큰 사용량, 비용($), 메시지 수, 세션 수
- **모델별** 토큰 분류 (Opus, Sonnet, Haiku)
- **세션별** 상세 내역 (시간, 토큰, 비용)
- **프로젝트별** 필터링
- **일별 차트** 추이 확인

### 👥 팀 모니터링

- 팀 생성/삭제, 멤버 추가/제거
- 에이전트 상태 실시간 감지 (활성/유휴/로그인 필요)
- 솔로 에이전트 생성 및 관리
- 팀 일괄 시작/재시작

### 🎮 Gamification

- 커밋, 세션, 메시지, 토큰으로 **XP** 획득
- 레벨업 시스템
- **24개 업적** 달성 (첫 커밋, 연속 출석, 백만 토큰 등)
- 연속 출석(스트릭) 추적

### 🔔 알림

- 에이전트 상태 변경 (작업 시작/종료)
- 새 커밋 감지
- 로그인 필요 경고
- 업적 달성 토스트

### 🔄 자동 업데이트

- 앱 시작 시 + 6시간마다 새 버전 체크
- 원클릭 다운로드 & 설치

## 설치

### 다운로드

[Releases](https://github.com/tripbtoz/ddabongdochi/releases/latest) 페이지에서 최신 `.zip` 파일을 다운로드합니다.

### 설치 방법

1. 다운로드한 `.zip` 파일 압축 해제
2. `ddanbongdochi.app`을 `/Applications` 폴더로 이동
3. 최초 실행 시 아래 **macOS 보안 설정** 참고

### macOS 보안 설정 (서명되지 않은 앱 설치)

이 앱은 Apple Developer 인증을 받지 않았기 때문에 macOS Gatekeeper가 실행을 차단합니다.

**방법 1: 우클릭으로 열기**
1. Finder에서 `ddanbongdochi.app` 우클릭 (또는 Control + 클릭)
2. **열기** 선택
3. "확인되지 않은 개발자" 경고에서 **열기** 클릭

**방법 2: 시스템 설정에서 허용**
1. 앱을 더블클릭 (차단됨)
2. **시스템 설정 → 개인정보 보호 및 보안** 이동
3. 하단에 "ddanbongdochi 앱이 차단되었습니다" 메시지 확인
4. **확인 없이 열기** 클릭

> ⚠️ 위 방법들은 최초 1회만 필요합니다. 이후 정상적으로 실행됩니다.

### 시스템 요구사항

| 항목 | 요구사항 |
|------|----------|
| OS | macOS 11.0 이상 |
| 아키텍처 | Apple Silicon / Intel |

## 사용법

### 실행 모드 선택

최초 실행 시 3가지 모드 중 선택:

| 모드 | 설명 |
|------|------|
| **메뉴바 전용** | 메뉴바 아이콘만 표시, 클릭 시 사용량 팝오버 |
| **메뉴바 + 교실** | 메뉴바 + 교실 창 동시 사용 **(권장)** |
| **교실 전용** | 교실 창만 표시, 팀 모니터링 집중 |

> 모드는 하단 바의 스위치 버튼으로 언제든 변경 가능

### 교실 화면

```
┌─────────────────────────────────────┐
│  [칠판 - 팀 메시지]    [게시판]      │
│                                     │
│  🦔  🦔  🦔  🦔   ← 팀원 캐릭터     │
│  ┌──┐┌──┐┌──┐┌──┐  ← 책상          │
│                                     │
│  ─────────────────────────────────  │
│  팀명 ● ● ●  | 4명 | 👥 🔄 ⇄      │  ← 하단 상태바
└─────────────────────────────────────┘
```

- **칠판**: 팀 채팅/메시지 표시 (마우스 휠로 스크롤)
- **캐릭터**: 활성 에이전트는 위아래로 움직임
- **하단 바**: 팀 상태 요약, 새로고침, 모드 전환

### 팀 관리

하단 바의 **👥 버튼**으로 팀 관리 패널 열기:

- **새 팀 생성**: 팀 이름 입력 → 생성
- **팀원 추가**: 팀 선택 → 이름/역할 입력 → 추가
- **솔로 에이전트**: 이름 + 작업 경로 입력 → 생성
- **팀 시작**: 시작 버튼으로 전체 에이전트 실행
- **팀/멤버 삭제**: 삭제 버튼 (확인 필요)

### 통계 확인

메뉴바 모드에서 **고슴도치 아이콘 클릭** 또는 교실 창의 사이드 패널에서:

- 오늘/주간 토큰, 비용, 메시지, 커밋 확인
- 모델별 사용량 분류
- 프로젝트별 필터링
- 세션 상세 내역

### OAuth 로그인 (선택)

Anthropic API 사용량 조회를 위해 OAuth 로그인 가능:

1. 설정에서 **로그인** 클릭
2. 브라우저에서 Anthropic 계정 인증
3. 앱으로 자동 복귀

> OAuth 없이도 로컬 JSONL 로그 기반 통계는 정상 동작합니다.

## 데이터 & 프라이버시

- 모든 데이터는 **로컬 Mac에만** 저장
- 세션 로그: `~/.claude/projects/` 읽기
- 게이미피케이션: `~/.claude/gamification.json`
- 외부 전송 없음 (GitHub 업데이트 체크 + OAuth 제외)

## 문제 해결

| 문제 | 해결 |
|------|------|
| 교실에 캐릭터가 안 보임 | 팀 관리에서 팀 생성 후 멤버 추가 |
| 토큰 데이터 없음 | `~/.claude/projects/`에 `.jsonl` 파일 존재 확인 |
| 업데이트 실패 | 인터넷 연결 확인 |
| 알림 안 옴 | macOS 설정 → 알림 → 따봉도치 허용 |
| "개발자를 확인할 수 없음" | 우클릭 → 열기 |
